# Datasets for CS6220 - Data Mining

This repository contains supporting datasets.

## Datasets

* [Iris Flower Dataset](https://archive.ics.uci.edu/ml/datasets/iris)
* [MovieLens 1M Dataset](https://grouplens.org/datasets/movielens/1m/)
* [Ames, Iowa Housing](http://www.amstat.org/publications/jse/v19n3/decock/AmesHousing.txt)
* [King County House Sales](https://www.kaggle.com/harlfoxem/housesalesprediction)
* [Online Retail](http://archive.ics.uci.edu/ml/machine-learning-databases/00352/Online%20Retail.xlsx)
